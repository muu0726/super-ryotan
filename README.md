# SUPER RYO-TAN

`キャラクター.png` を主人公にした、HTML5 Canvas + Vanilla JavaScript (Vite) 製のマリオ風2D横スクロールアクションゲーム。

## 特徴

- **マリオ物理の再現**: 慣性・ブレーキスリップ・可変ジャンプ(長押しで高く、離すと失速)・落下時2倍重力・終端速度
- **蓄積型 Fixed Timestep**: 16.67ms 固定の物理更新 + 描画補間で、負荷変動時も挙動が安定
- **全31ステージ (1〜30 + EX)**: 文字列テンプレートによるマップ定義。番号が進むほど難しくなる設計で、クリア可能性は物理シミュレーションテストで検証済み。ゴール到達で次ステージが解放され、`localStorage` に保存
- **モバイル対応**: マルチタッチ仮想ボタン(同時押し対応・ヒットエリア1.75倍)、縦向き時は回転促しオーバーレイ
- **効果音**: Web Audio API (OscillatorNode) による動的生成。外部音声ファイルなし

## 操作

| 操作 | キーボード | タッチ |
| --- | --- | --- |
| 移動 | ← → / A D | ◀ ▶ |
| ジャンプ | Z / SPACE / ↑ | A ボタン |
| ダッシュ | X / SHIFT(ホールド) | B ボタン(タップで切替) |

キーボードのダッシュは押している間だけ有効、タッチの B ボタンはタップごとの ON/OFF 切替です。
ダッシュ中は HUD に「» DASH」が表示され、B ボタンが点灯します。

`?stage=N` を URL に付けると指定ステージから直接開始できます(動作確認用)。

## 開発

```bash
npm install
npm run dev      # 開発サーバ
npm run build    # dist/ に本番ビルド
npm run preview  # ビルド結果の確認
```

## デプロイ (Vercel)

`vercel.json` 設定済み。リポジトリを GitHub に push し、Vercel でインポートするだけでデプロイされます
(Build Command: `npm run build` / Output: `dist` は自動検出)。
